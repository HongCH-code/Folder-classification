# 📷 照片日期分類工具

一個專業的照片管理工具，採用攝影師作品集風格設計，自動根據照片拍攝日期分類並整理到不同資料夾。

## ✨ 功能特色

### 核心功能
- 🗓️ **智能日期提取** - 自動從照片 EXIF 資料讀取拍攝日期
- 📁 **靈活分類選項** - 支援多種日期格式分類方式
- 🎯 **階層式組織** - 支援多層級資料夾結構（年/月/日）
- 💾 **雙模式操作** - 可選擇複製或移動檔案
- 📊 **即時進度追蹤** - 視覺化進度條和詳細日誌
- 🎨 **專業 UI/UX** - 基於攝影作品集風格的現代化介面

### 分類方式

**單層分類：**
- 年-月-日（例：2024-01-15）
- 年-月（例：2024-01）
- 年（例：2024）

**多層分類：**
- 年 → 年-月 → 年-月-日（三層結構）
- 年 → 年-月（二層結構）
- 年-月 → 年-月-日（二層結構）

### 支援格式
`.jpg` `.jpeg` `.png` `.gif` `.bmp` `.webp` `.heic` `.heif`

## 🚀 使用方法

### 1. 開啟工具
在支援的瀏覽器中打開 `index.html`

### 2. 選擇照片資料夾
點擊「選擇照片資料夾」按鈕，選擇包含照片的資料夾

### 3. 設定分類選項
- 選擇資料夾分類方式（年/月/日組合）
- 選擇是否使用複製模式（保留原檔案）
- 選擇是否在原資料夾內創建子資料夾

### 4. 自動處理
工具會自動：
- 掃描所有照片檔案
- 讀取 EXIF 拍攝日期
- 建立相應的資料夾結構
- 複製或移動照片到對應資料夾
- 顯示處理進度和結果

## ⚙️ 設定選項

### 分類方式
選擇照片資料夾的命名格式

### 複製模式
- **開啟** - 保留原始照片，複製到新資料夾
- **關閉** - 移動照片到新資料夾（刪除原位置）

### 子資料夾創建
- **開啟** - 在原資料夾內創建分類子資料夾
- **關閉** - 選擇其他目標資料夾

## 🔒 隱私與安全

- ✅ 所有處理均在本地瀏覽器中完成
- ✅ 不會上傳任何照片到伺服器
- ✅ 不會收集或傳送使用者資料
- ✅ 完全離線可用
